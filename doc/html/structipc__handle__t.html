struct ipc_handle_t*(* ipc_handle_t::accept) (struct ipc_handle_t *server_handle)

Accepts a new connection on a server socket.

This function pointer is used for server socket IPC to accept incoming client connections. When implemented, it should accept a new connection from the server socket, creating and returning a new IPC handle for the client connection.

Parameters
server_handlePointer to the server IPC handle. This handle represents the server socket that is listening for incoming connections.
Returns
A pointer to a new ipc_handle_t representing the client connection on success. Returns NULL if the connection could not be accepted or an error occurred.

int(* ipc_handle_t::destroy) (void *ctx)

Destroys the IPC mechanism.

This function cleans up any resources allocated by the IPC mechanism, effectively terminating its use. It should be called when the IPC mechanism is no longer needed.

Parameters
ctxcontext pointer for different IPCs to the function pointers
Returns
0 on success, or a negative error code on failure.
